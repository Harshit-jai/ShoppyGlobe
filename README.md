# ShoppyGlobe - React E-commerce App

## How to Run
1. Clone the repository:
   ```
   git clone https://github.com/Harshit-jai/shoppyglobe.git
   cd shoppyglobe
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Run the app:
   ```
   npm run dev
   ```

4. Open browser at http://localhost:5173/
